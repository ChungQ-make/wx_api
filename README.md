## 基于node和express框架的搭建的二手市场后端api服务器  
+ 包括前端服务模块和管理员模块
+ 前端服务前后端是分离模式开发
+ 管理员模块是服务端渲染模式开发

## API 基址
+ 前端服务
~~~
// 测试基址
http://localhost:5000/api/public/v1

// https基址
https://www.yycloud.ltd/api/public/v1
~~~

+ 管理员服务

~~~
// 测试基址
http://localhost:5000/api/private/v1

// https基址
https://www.yycloud.ltd/api/private/v1
~~~
